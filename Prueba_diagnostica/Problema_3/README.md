## Problema 3: Reconocimiento de Cadenas — Notación Científica, Dirección IP y Correo Electrónico

## Descripción

El objetivo de este programa es reconocer automáticamente el tipo de cadena ingresada por el usuario, determinando si corresponde a una de las siguientes categorías:

Notación científica (por ejemplo: 1.23e10, -3E-5, 2e+8)

Dirección IP válida (por ejemplo: 192.168.1.1, 255.255.255.0)

Correo electrónico (por ejemplo: usuario@gmail.com, nombre.apellido@dominio.edu)

El programa utiliza expresiones regulares (regex) para analizar y validar la cadena introducida.
