# Problema 2: Generación del Polinomio (x + 1)ⁿ usando el Triángulo de Pascal

## Descripción
El objetivo de este programa es **generar los coeficientes del polinomio (x + 1)ⁿ** utilizando el **Triángulo de Pascal**, mostrar el desarrollo paso a paso de la función  
**f(x) = (x + 1)ⁿ**, y **comparar los tiempos de ejecución** entre implementaciones en **C** y **Python**.

El algoritmo implementa **memoria dinámica** (en C) y escribe los resultados en un archivo `resultado.txt`.

## Requisitos de instalación
Instalar un compilador compatible (como GCC):
Instalar Python 3 (versión 3.8 o superior).
