# Programa: Traducción de Palabras Reservadas en C

## Descripción
Este programa analiza un archivo de código fuente en **C** cargado dinámicamente en memoria y **detecta las palabras reservadas** del lenguaje C, mostrando su **traducción al español**.  

El algoritmo utiliza **memoria dinámica** para almacenar el contenido del archivo y recorre el código para identificar los tokens correspondientes a palabras reservadas.

## Palabras reservadas traducidas
Algunas de las palabras reservadas que el programa traduce:
auto -> automático
break -> romper
case -> caso
char -> carácter
const -> constante
continue -> continuar
default -> predeterminado
